##  Tech Stack
| Layer | Technology |
|---|---|
| Language | C# (.NET Core) |
| ORM / Data Access | Dapper (micro-ORM) |
| Database | SQL Server — relational schema with T-SQL stored procedures |
| Auth | Custom cookie authentication (`Microsoft.AspNetCore.Authentication.Cookies`) + `Microsoft.AspNetCore.Identity.PasswordHasher<T>` for credential hashing — Dapper-backed, no EF Core |
| Export | PDF and XML downloads on listing views |
| Frontend | Server-rendered Razor views with custom CSS |
