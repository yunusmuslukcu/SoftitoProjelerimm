## 🛠️ Tech Stack
* **C# .NET Core**
* **ASP.NET Core MVC**
* **Entity Framework Core (Code-First)**
* **ASP.NET Core Identity** (EF Core store, cookie-based authentication, account lockout policy, token-based password reset)
* **T-SQL**
* **SweetAlert**
