## 🚀 Tech Stack & Specifications

- **Framework & Language:** .NET 10 (ASP.NET Core MVC) & C# 14
- **Database & ORM:** MS SQL Server with Entity Framework Core 10 (Code-First & Migrations)
- **Architecture:** Generic Repository Pattern, Dependency Injection (DI), Separation of Concerns (SoC)
- **Security & Auth:** Modular Areas (Customer & Staff), Session-Based Authentication, Custom Action Filters for authorization
- **Optimization:** In-Memory Caching (`IMemoryCache`) for fast catalog rendering
- **Reporting:** `ClosedXML` for dynamic Excel exports, `QuestPDF` for pixel-perfect PDF generation
- **Logging & Auditing:** Custom asynchronous 
- **Frontend / UI:** Bootstrap 5.3, Bootstrap Icons, responsive neo-brutalist / retro-cream design, custom hover scaling  dynamic JS password toggles
